Nimbus keeps the machine as the source of truth: a local SQLite index, MCP connectors, and a structural HITL consent gate.

:::note[Documentation tip]
This site tracks the **repository `main` branch** (unreleased). Versioned snapshots are added at release tags; until then, cross-check critical behaviour with the root `docs/` tree in your checkout.
:::

Use the sidebar to open **Getting started** for install and first-run steps on Windows, macOS, and Linux.

## More documentation

The canonical **roadmap**, **Phase 3.5 plan**, **architecture**, and **security** docs live in the `docs/` folder at the repository root (next to `packages/`). Open that tree in your checkout — this Starlight site will grow to mirror those sources over time.
